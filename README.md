### 特性

- 基于Spring boot开发

- 快速部署

### 原理说明

本项目基于 [yihleego](https://github.com/yihleego) 的 [banana](https://github.com/yihleego/banana) 项目，实现了在网页上在线生成的功能，使得获取想要的ASCII字符更方便快捷。

### 下载安装

git克隆后使用idea打开即可

### 使用方法

在线使用或下载代码后二次开发
